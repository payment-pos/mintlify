title: "Plan (/me/plan)"
description: "Kuruluşunuzun atanmış planı ve plana dahil feature'lar."
---

`/me/plan` endpoint'i; aktif kuruluşunuzun atanmış planını ve plana dahil feature'ları döner. Sadece okunabilir; plan değişikliği konsoldan veya satış ekibinden talep edilir.

## Endpoint

```http
GET /api/v1/me/plan
```

JWT Bearer auth gerektirir.

## İstek

```bash
curl https://identity.payven.com.tr/api/v1/me/plan \
  -H "Authorization: Bearer $PAYVEN_TOKEN"
```

## Yanıt

`200 OK` — `MyPlanDto`:

```json
{
  "plan": {
    "id":            "abc-...",
    "code":          "standard",
    "name":          "Standart Plan",
    "description":   "Sanal POS + Para Transferi modülleri",
    "price":         999900,
    "currency":      "TRY",
    "billing_cycle": "monthly",
    "features": [
      {
        "code":     "sanalpos",
        "name":     "Sanal POS",
        "category": "module"
      },
      {
        "code":     "para-transferi",
        "name":     "Para Transferi",
        "category": "module"
      },
      {
        "code":     "fraud",
        "name":     "Fraud Yönetimi",
        "category": "addon"
      }
    ]
  }
}
```

| Alan | Açıklama |
|---|---|
| `plan.id` | Plan kimliği |
| `plan.code` | Plan kodu (`free`, `standard`, `enterprise`, vb.) |
| `plan.name`, `description` | İnsan-okur ad |
| `plan.price` | Kuruş cinsinden (varsayılan TRY) |
| `plan.billing_cycle` | `monthly`, `yearly` |
| `plan.features[]` | Plana dahil feature'lar — `category: "module"` aktif modül, `addon` ek paket |

Kuruluşa henüz plan atanmamışsa yanıt `null` döner.

## Aktif feature'ları öğrenmek

Plana dahil feature'ların **bu kullanıcı için aktif olup olmadığını** öğrenmek için [`GET /me`](/identity/auth/me) endpoint'ini kullanın — orada `features[].is_active` alanı kullanıcı + kuruluş + plan üçlüsünden filtrelenmiş halde döner.

## Yol haritası

- **Kullanım sayaçları** — bu plan kapsamında aylık işlem sayısı, RPS limiti, aktif merchant sayısı (henüz yanıtta yer almıyor)
- **Plan kullanım webhook'ları** — `plan.usage.warning`, `plan.usage.exceeded` (planlanıyor)
- **Self-service plan yükseltme** — şu an konsoldan veya satış kanalı üzerinden
