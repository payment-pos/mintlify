Bu sayfa, Payven API'sinin tüm versiyonlarındaki breaking olmayan ve breaking change'leri kronolojik olarak listeler.

<Update label="2026-05-03" description="v1">
  ### Yeni
  - Sanal POS modülü için `POST /payments/order-link` (Pay-by-Link) endpoint'i eklendi.
  - `/api/v1/checkout/sessions` Hosted Checkout endpoint'leri yayınlandı.

  ### İyileştirme
  - Akıllı Yönlendirme: bileşik skor hesabına banka sağlığı (latency, hata oranı) parametreleri dahil edildi.
  - Smart Retry maksimum deneme sayısı kural bazında konfigüre edilebilir hale geldi.
  - Hosted Checkout sayfası `appearance` alanıyla görsel özelleştirme destekli.

  ### Düzeltme
  - 3DS sonrası `return_url` query parametrelerinde Türkçe karakter encoding sorunu giderildi.
</Update>

<Update label="2026-04-15" description="v1">
  ### Yeni
  - DCC (Dynamic Currency Conversion) akışı yayınlandı.
  - Mutabakat için `bulk-resolve` endpoint'i eklendi — toplu tutarsızlık çözümü.

  ### İyileştirme
  - Webhook teslim retry takvimi optimize edildi (1m → 5m → 30m → 2h → 24h).
  - API anahtarı listeleme yanıtında `last_used_at` alanı eklendi.
</Update>

<Update label="2026-03-20" description="v1">
  ### Yeni
  - Pre-Auth + Capture akışı için `operation_type: "pre_auth"` desteği.
  - Konnektör Sağlık Paneli (Konsol).

  ### İyileştirme
  - Idempotency-Key saklama süresi 12 saatten 24 saate çıkarıldı.
</Update>
