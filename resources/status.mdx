Tüm Payven servislerinin gerçek zamanlı durumunu kontrol etmek için:

[**status.payven.com.tr →**](https://status.payven.com.tr)

## İzlenen servisler

| Servis | Açıklama |
|---|---|
| Identity & Auth | Kimlik doğrulama, API anahtarı yönetimi |
| Sanal POS API | Ödeme alma, iade, sorgulamalar |
| Sanal POS Webhook | Webhook teslim altyapısı |
| Hosted Checkout | Müşteri tarafı ödeme sayfası |
| Konsol | Dashboard web uygulaması |
| Banka Konnektörleri | Her bir banka entegrasyonu için ayrı durum göstergesi |

## Bildirim aboneliği

Status sayfasından e-posta veya RSS aboneliği yapabilirsiniz. Olay açıldığında ve kapandığında bildirim alırsınız.

## Geçmiş

90 günlük olay geçmişi durum sayfasında görüntülenir. Her olay için:

- Etkilenen servisler
- Başlangıç/bitiş zamanı
- Müşteri etkisi
- Kök neden analizi (post-mortem, kritik olaylar için)

bilgileri sunulur.
