Payven Konsolu (Dashboard), tüm Payven ürünlerini tek bir arayüzden yönetebileceğiniz web uygulamasıdır.

[**Konsola Git →**](https://dashboard.payven.com.tr)

## Konsol kullanımı

| Kategori | Ekranlar |
|---|---|
| **İşlemler** | İşlem listesi, detay, timeline, iade, iptal, export |
| **Bayiler** | Merchant CRUD, alt bayi yönetimi, vergi/iletişim bilgileri |
| **Konnektörler** | Banka tipi katalogu, yetenek matrisi |
| **Konfigürasyonlar** | Banka credential yönetimi, bağlantı testi |
| **Akıllı Yönlendirme** | Kural CRUD, simülasyon, sıralama |
| **Mutabakat** | Gün sonu mutabakatı, tutarsızlık çözümü |
| **API Anahtarları** | Anahtar oluşturma, IP whitelist, rotasyon |
| **Webhook'lar** | Abonelik yönetimi, teslim kayıtları, manuel yeniden gönderme |
| **Hata Kodları** | Tüm Payven ve banka kodları |
| **Sistem Sağlığı** | Servis durumu, banka sağlık paneli, gerçek zamanlı metrikler |
| **Kullanıcılar** | Kullanıcı + rol yönetimi |
| **Plan ve Kullanım** | Aktif plan, kotalar, faturalama |

## Mobil uyumluluk

Konsol mobil cihazlarda kullanılabilir — sidebar gizlenir, tablo sütunları otomatik daraltılır.
