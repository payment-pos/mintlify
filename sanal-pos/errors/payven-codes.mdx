Bu sayfa Sanal POS'un kendi domain hata kodlarını listeler. Bu kodlar `PaymentOperationResult.error_code` alanında ve hata zarfının `code` alanında döner. Banka tarafından gelen ham yanıt kodları için: [Banka Yanıt Kodları](/sanal-pos/errors/bank-codes).

## Genel ödeme akışı (`PV_*`)

| Kod | Anlam |
|---|---|
| `PV_INSUFFICIENT_FUNDS` | Yetersiz bakiye |
| `PV_CARD_EXPIRED` | Kart süresi dolmuş |
| `PV_INVALID_CARD` | Kart numarası geçersiz (Luhn / format) |
| `PV_INVALID_CVV` | CVV yanlış |
| `PV_INVALID_CARD_NUMBER` | Kart numarası alan validasyonu başarısız |
| `PV_INVALID_CARD_EXPIRY` | Son kullanma tarihi format hatalı |
| `PV_INVALID_CARD_HOLDER` | Kart sahibi adı geçersiz / boş |
| `PV_INVALID_AMOUNT` | Tutar geçersiz (negatif, sıfır, minimum altında) |
| `PV_INVALID_CURRENCY` | Desteklenmeyen para birimi |
| `PV_INVALID_INSTALLMENT` | Taksit sayısı bu kart için desteklenmiyor |
| `PV_BANK_DECLINED` | Banka işlemi reddetti — `provider_error_code` daha spesifik bilgi taşır |
| `PV_DO_NOT_HONOR` | Banka spesifik "Do Not Honor" reddi (kod 05) |
| `PV_LIMIT_EXCEEDED` | Banka harcama limiti aşıldı |
| `PV_FRAUD_SUSPECTED` | Banka fraud şüphesiyle reddetti |
| `PV_BIN_RESTRICTED` | BIN kara liste / kısıtlama nedeniyle reddedildi |
| `PV_DUPLICATE_ORDER` | Aynı `external_id` ile başka bir işlem zaten var |
| `PV_PAYMENT_FAILED` | Genel ödeme başarısız (spesifik kod yok) |
| `PV_PAYMENT_EXCEPTION` | Connector'a ulaşılamadan yakalanan exception |
| `PV_TIMEOUT` | Banka yanıt vermedi |
| `PV_SYSTEM_ERROR` | Payven iç sistem hatası |
| `PV_CONFIG_ERROR` | Konnektör konfigürasyonu hatalı / eksik |
| `PV_UNKNOWN` | Eşlenmemiş hata — destek talebine açın |

## Akış kontrol kodları

| Kod | Anlam |
|---|---|
| `PV_TRANSACTION_NOT_FOUND` | Verilen `transaction_id` bulunamadı |
| `PV_CONNECTOR_NOT_FOUND` | Bu işlem için uygun konnektör yok |
| `PV_ROUTING_RULE_NOT_FOUND` | Hiçbir yönlendirme kuralı koşulları karşılamadı |
| `PV_INVALID_MERCHANT_ID` | `X-Merchant-Id` veya `X-External-Merchant-Id` header'ı geçersiz |
| `PV_MERCHANT_NOT_FOUND` | Merchant DB'de bulunamadı |
| `PV_MERCHANT_INACTIVE` | Merchant pasif statüde |
| `PV_INVALID_BILLING_CITY` | Fatura adresi şehir kodu geçersiz |
| `PV_INVALID_SHIPPING_CITY` | Gönderim adresi şehir kodu geçersiz |
| `PV_INVALID_CALLBACK_URL` | `callback_url` HTTPS değil veya geçersiz |
| `PV_NON_3D_NOT_ALLOWED` | Tenant policy'sinde Non-3D ödeme kapalı |
| `PV_EMPTY_CALLBACK_DATA` | Banka callback'i boş veya parse edilemedi |
| `PV_MISSING_TRANSACTION_ID` | Callback'te transaction_id yok |

## 3D Secure kodları

| Kod | Anlam |
|---|---|
| `PV_3DS_FAILED` | 3DS doğrulaması başarısız (genel) |
| `PV_3DS_INIT_EXCEPTION` | 3DS başlatma sırasında exception |
| `THREEDS_NOT_SUPPORTED` | Konnektör 3DS desteklemiyor |
| `THREEDS_INIT_NOT_ALLOWED` | İşlem mevcut durumunda 3DS init yapılamaz |
| `THREEDS_VERIFY_FAILED` | 3DS verify adımı başarısız (CAVV/ECI mismatch) |
| `THREEDS_COMPLETE_NOT_ALLOWED` | 3DS complete bu durumda izin verilmiyor |
| `THREEDS_COMPLETE_FAILED` | 3DS complete sırasında banka reddi |
| `THREEDS_COMPLETE_EXCEPTION` | 3DS complete sırasında exception |
| `THREEDS_CARD_DATA_MISSING` | Callback'te kart verisi eksik |
| `THREEDS_MODULE_NOT_FOUND` | 3DS modülü konnektörde implemente değil |
| `THREEDS_TIMEOUT` | 3DS akışı 15 dakika içinde tamamlanmadı |
| `CARD_MISMATCH` | Init ile complete arasında kart bilgisi farklı |

## Generic / HTTP canonical kodları

Aşağıdaki kodlar HTTP altyapısı tarafından üretilir; spesifik bir Payven domain kodu yoksa fallback olarak kullanılır. Tüm bu kodlar **lowercase snake_case** formattadır.

| HTTP | `code` | Anlam |
|---|---|---|
| `400` | `bad_request` | Genel istek hatası |
| `400` | `invalid_json` | Body geçersiz JSON |
| `401` | `unauthorized` | Token eksik / geçersiz |
| `403` | `forbidden` | Yetki yok |
| `404` | `resource_not_found` | Generic kaynak bulunamadı |
| `409` | `conflict` | Generic çakışma |
| `409` | `idempotency_key_in_use` | Aynı `Idempotency-Key` farklı body için kullanılmış |
| `422` | `validation_failed` | FluentValidation kuralı ihlali |
| `429` | `rate_limit_exceeded` | İstek limiti aşıldı |
| `500` | `internal_server_error` | Sunucu hatası |
| `502` | `bad_gateway` | Upstream / connector kötü yanıt |
| `503` | `service_unavailable` | Servis geçici olarak erişilemez |

## Bilinmeyen kod

`PV_UNKNOWN` — connector'dan gelen kod Payven eşleme tablosunda yok. Bu durumlarda ham mesaj `provider_error_code` ve `message` alanlarında bulunur. Lütfen `correlation_id` ile birlikte [destek ekibimize](/resources/support) yazın — eşleme tablosunu güncelleyebiliriz.
